# Compare Information Criteria of Different Models

Compare information criteria of different models fitted with
[`waic`](https://paulbuerkner.com/brms/reference/waic.brmsfit.md) or
[`loo`](https://paulbuerkner.com/brms/reference/loo.brmsfit.md).
Deprecated and will be removed in the future. Please use
[`loo_compare`](https://paulbuerkner.com/brms/reference/loo_compare.brmsfit.md)
instead.

## Usage

``` r
compare_ic(..., x = NULL, ic = c("loo", "waic", "kfold"))
```

## Arguments

- ...:

  At least two objects returned by
  [`waic`](https://paulbuerkner.com/brms/reference/waic.brmsfit.md) or
  [`loo`](https://paulbuerkner.com/brms/reference/loo.brmsfit.md).
  Alternatively, `brmsfit` objects with information criteria precomputed
  via [`add_ic`](https://paulbuerkner.com/brms/reference/add_ic.md) may
  be passed, as well.

- x:

  A `list` containing the same types of objects as can be passed via
  `...`.

- ic:

  The name of the information criterion to be extracted from `brmsfit`
  objects. Ignored if information criterion objects are only passed
  directly.

## Value

An object of class `iclist`.

## Details

See
[`loo_compare`](https://paulbuerkner.com/brms/reference/loo_compare.brmsfit.md)
for the recommended way of comparing models with the loo package.

## See also

[`loo`](https://paulbuerkner.com/brms/reference/loo.brmsfit.md),
[`loo_compare`](https://paulbuerkner.com/brms/reference/loo_compare.brmsfit.md)
[`add_criterion`](https://paulbuerkner.com/brms/reference/add_criterion.md)
