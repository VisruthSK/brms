# Model comparison with the loo package

For more details see
[`loo_compare`](https://mc-stan.org/loo/reference/loo_compare.html).

## Usage

``` r
# S3 method for class 'brmsfit'
loo_compare(x, ..., criterion = c("loo", "waic", "kfold"), model_names = NULL)
```

## Arguments

- x:

  A `brmsfit` object.

- ...:

  More `brmsfit` objects.

- criterion:

  The name of the criterion to be extracted from `brmsfit` objects.

- model_names:

  If `NULL` (the default) will use model names derived from deparsing
  the call. Otherwise will use the passed values as model names.

## Value

An object of class "`compare.loo`".

## Details

All `brmsfit` objects should contain precomputed criterion objects. See
[`add_criterion`](https://paulbuerkner.com/brms/reference/add_criterion.md)
for more help.
