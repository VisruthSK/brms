# Number of Grouping Factor Levels

Extract the number of levels of one or more grouping factors.

## Usage

``` r
# S3 method for class 'brmsfit'
ngrps(object, ...)

ngrps(object, ...)
```

## Arguments

- object:

  An R object.

- ...:

  Currently ignored.

## Value

A named list containing the number of levels per grouping factor.
