# Rename parameters in brmsfit objects

Rename parameters within the `stanfit` object after model fitting to
ensure reasonable parameter names. This function is usually called
automatically by [`brm`](https://paulbuerkner.com/brms/reference/brm.md)
and users will rarely be required to call it themselves.

## Usage

``` r
rename_pars(x)
```

## Arguments

- x:

  A `brmsfit` object.

## Value

A `brmsfit` object with adjusted parameter names.

## Details

Function `rename_pars` is a deprecated alias of `rename_pars`.
